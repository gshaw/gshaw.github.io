# Gerry's Personal Home Page

Jekyll site for Gerry's personal home page.

[Live Site](https://gshaw.ca)

### Build Instructions

```sh
brew install mise
mise install
mise run install
mise run dev
mise run deploy
```

Pushing to GitHub will publish the site.
